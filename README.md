# Proyecto Gimnasio

Proyecto desarrollado con Spring Boot para la gestión de entrenadores y clases de gimnasio.

## Tecnologías utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- H2 Database

## Funcionalidades

- CRUD de entrenadores
- CRUD de clases
- Relación OneToMany / ManyToOne
- API REST
- Interfaz web con Thymeleaf
- Filtros por día y nivel

## URLs principales

- http://localhost:8080
- http://localhost:8080/web/entrenadores
- http://localhost:8080/web/clases
- http://localhost:8080/api/entrenadores
- http://localhost:8080/api/clases

## Base de datos H2

- http://localhost:8080/h2-console
